# SimplePlayer
SimplePlayer یک پلاگین ساده و سبک برای تایپ‌چو است که با استفاده از کتابخانه APlayer، امکان پخش موسیقی را در نوشته‌ها فراهم می‌کند. با یک شورت‌کد ساده می‌توانید لیست پخش دلخواه خود را ایجاد کرده و آهنگ‌ها را به همراه عنوان، نام خواننده و تصویر کاور نمایش دهید. این پلاگین از زبان فارسی پشتیبانی کامل داشته و ظاهری واکنش‌گرا و زیبا دارد.
